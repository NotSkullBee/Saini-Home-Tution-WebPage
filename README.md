# Saini Home Tuition Website

This project is a high-end, fully responsive website designed for a home tuition service based in Patna, Bihar. The goal was to create a premium, conversion-focused interface that builds trust and encourages direct user action (calls, WhatsApp, social engagement).

## Key Features

- Modern luxury UI with glassmorphism and soft gradient design
- Fully mobile-optimized layout (primary user base)
- Smooth animations and scroll-triggered counters
- Side navigation menu (mobile-first UX)
- Floating call-to-action button for instant contact
- WhatsApp and direct call integration
- Social media integration (Instagram & YouTube)
- Structured program and fee sections
- Embedded demo class video (on-click playback)
- Clean typography and spacing for readability

## Tech Stack

- HTML5
- CSS3 (Glassmorphism, Flexbox, Responsive Design)
- Vanilla JavaScript (Animations, Interactions)

## Objective

To simulate a premium ₹1–5 lakh coaching institute website experience using only front-end technologies, focusing on:
- User trust
- Clean design
- High conversion rate
